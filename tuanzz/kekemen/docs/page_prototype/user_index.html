豉汁培根金针卷

1、牛肉切片再切成粗条，用少许生粉、糖、黑胡椒、老抽，鸡粉腌味半小时候以上，干葱、姜、蒜切片。

2、茄子一个，切成跟牛肉大小一样，红萝卜半条也切成大小一样的条待用。

3、锅烧热放油，放牛肉煎时在牛肉里加入少许油容易松开牛肉，煎两面九成熟捞出待用。

4、锅里再放油、倒入杂菜炒软拔开一边、再放干葱蒜片、煎香、炒匀、洒入料酒、加盐、鸡粉、黑胡椒粒、炒匀后杂菜熟后，再倒入牛肉条炒匀即可出锅。

4、锅里再放油、倒入杂菜炒软拔开一边、再放干葱蒜片、煎香、炒匀、洒入料酒、加盐、鸡粉、黑胡椒粒、炒匀后杂菜熟后，再倒入牛肉条炒匀即可出锅。
